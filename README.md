# PC / CONFIGURATOR

Профессиональный конфигуратор персональных компьютеров с инженерной точностью. Проект разработан для быстрого подбора оптимальных игровых и рабочих станций на основе актуальных бенчмарков и характеристик комплектующих.

##  Основные возможности
- **Интерактивный каталог:** 11 готовых конфигураций — от офисных решений до экстремальных игровых станций.
- **Интеллектуальный подбор (Test):** Система тестирования, которая анализирует задачи пользователя и бюджет, предлагая лучший вариант.
- **Бенчмарки:** Визуализация производительности (FPS и баллы) для каждой сборки.
- **Engineering Design:** Минималистичный и строгий интерфейс, вдохновленный технической документацией и терминалами.

##  Технологии
- **React 18** + **Vite** (для мгновенной сборки и работы)
- **React Router Dom (HashRouter)** — стабильная навигация, адаптированная под GitHub Pages.
- **CSS Modules / Grid / Flexbox** — точная сетка и адаптивность.

##  Проектирование и аналитика (Ветка Task)
Разработка проекта велась по канонам системного анализа. Полная документация доступна в ветке `task`:

* **`task.md`** — Техническое задание и план разработки.
* **`userFlow.md`** — Карта путей пользователя в приложении.
* **`pc-configuration.pdf`** — Сводные таблицы и визуальные схемы сборок.
* **`grow.md`** / **`optimize.md`** — Стратегия развития и планы по оптимизации кода.
* **`terms.md`** — Глоссарий используемых терминов.
* **`review.md`** / **`remark.md`** — Отчеты о тестировании и заметки по UI/UX.

##  Инструкция по запуску локально

1. **Клонируйте репозиторий:**
   ```bash
   git clone [https://github.com/Flu337/PC-Configurator.git](https://github.com/Flu337/PC-Configurator.git)

2. **Перейдите в папку проекта:**
   ```bash
   cd PC-Configurator

3. **Установите зависимости:**
   ```bash
   npm install

4. **Запустите сервер разработки:**
   ```bash
   npm run dev

# PR-стратегия PC-Configurator: привлечение пользователей

PR-стратегия проекта PC-Configurator направлена на привлечение геймеров и профессионалов, повышение доверия к качеству сборок и формирование имиджа экспертного сервиса в сфере компьютерного железа. 
Основная цель — создать поток входящих заявок через демонстрацию прозрачности тестов и уникального визуального стиля.

## 1. Целевая аудитория

Основная аудитория проекта:
* **Геймеры и энтузиасты:** пользователи, которым важен FPS и эстетика сетапа.
* **Creative Pro:** видеомонтажеры, дизайнеры и архитекторы, нуждающиеся в стабильных рабочих станциях.
* **Студенты технических вузов:** аудитория, ценящая баланс цены и производительности.
* **Начинающие стримеры:** пользователи, ищущие готовые решения «под ключ» для трансляций.

Эти пользователи активно используют следующие платформы:
* YouTube (обзоры железа)
* Twitch (стриминг)
* Discord (сообщества по играм)
* Telegram (техно-каналы)
* VK (паблики по компьютерному железу)

## 2. Каналы продвижения

### 2.1 Органические каналы

**YouTube и Shorts / TikTok**
* Публикация ASMR-видео процесса сборки (от распаковки процессора до кабель-менеджмента).
* Тесты производительности в новых игровых тайтлах (Benchmark-шоу).
* Видео-ответы на частые вопросы по подбору комплектующих.

**VK и Telegram**
* Создание сообщества, где публикуются:
    * Фотосеты готовых заказов в стиле «Tech-noir».
    * Сравнение видеокарт и процессоров разных поколений.
    * Публичный лог выполненных работ и отзывов.

**Профильные форумы и порталы (Overclockers, Habr)**
* Публикация кейсов по решению сложных технических задач (например, оптимизация охлаждения в компактном корпусе).
* Статьи о настройке систем для работы под Linux.

### 2.2 Партнёрские каналы

* **Локальные компьютерные клубы:** брендирование зон или предоставление сборок для турниров.
* **Магазины периферии:** совместные акции (скидка на сборку при покупке девайсов партнера).

## 3. Контент-маркетинг

Контент должен подтверждать статус «Мастера». Планируется создание следующих типов материалов:

* **Технические отчеты (Logs):**
    * Публикация скриншотов стресс-тестов (AIDA64, FurMark) для каждой сборки.
    * Анализ температурных показателей.
* **Гайды по выбору:**
    * «Как не переплатить за бренд при выборе БП».
    * «Топ-5 сборок для работы в Blender 2026».
* **Визуальный контент:**
    * Качественные макро-фотографии компонентов.
    * Схемы циркуляции воздуха в корпусах.

## 4. Механики привлечения пользователей

* **Программа «Приведи друга»:** Бонусы или бесплатное годовое обслуживание (чистка + замена термопасты) за рекомендацию.
* **Бесплатный аудит:** Консультация по апгрейду текущего ПК пользователя, которая плавно подводит к заказу новой сборки.
* **Конфигуратор с сохранением:** Возможность собрать свою конфигурацию на сайте и получить экспертную оценку мастера в один клик.

## 5. Работа с сообществом

Для развития проекта критически важна «сарафанка»:
* **Сбор обратной связи:** Публикация скриншотов переписок с довольными клиентами.
* **Интерактивы:** Опросы типа «Intel или AMD: что выберете для работы?».
* **Open Source документация:** Публикация части наработок (например, гайдов по настройке Ubuntu) в открытый доступ.

## 6. Метрики эффективности

Для оценки PR-стратегии используются следующие показатели:
* **Количество прямых обращений** в мессенджеры через сайт.
* **Конверсия** из просмотра страницы товара в нажатие кнопки «Связаться».
* **Количество переходов** по ссылкам из соцсетей.
* **Цитируемость:** упоминание бренда в локальных чатах и сообществах.
* **Удержание (Retention):** количество повторных обращений за обслуживанием.

**Итог:** PR-стратегия PC-Configurator строится на экспертном контенте и визуальном превосходстве. Мы не просто продаем «железо», мы продаем уверенность в результате и профессиональный подход к каждой детали.

# PC / CONFIGURATOR

Профессиональный конфигуратор персональных компьютеров с инженерной точностью. Проект создан для подбора оптимальных игровых и рабочих станций с учетом актуальных бенчмарков.

##  Функционал
- **Интерактивный каталог:** 11 уникальных сборок — от бюджетных до экстремальных.
- **Интеллектуальный подбор:** Интерактивный тест, который подберет ПК под ваш бюджет и задачи.
- **Бенчмарки:** Реальные показатели производительности (FPS в играх и баллы в софте) для каждой системы.
- **Инженерный дизайн:** Минималистичный интерфейс, вдохновленный чертежами, терминалами и эстетикой чистой разработки.

##  Технологии
- **React 18** + **Vite** (сверхбыстрая сборка проекта)
- **React Router Dom** (бесшовная навигация между страницами)
- **CSS Grid/Flexbox** (адаптивная и точная верстка)

##  Документация и проектирование (Ветка Task)
В этой ветке сосредоточена вся аналитическая база проекта. Каждый файл описывает конкретный этап жизненного цикла разработки:

* **`task.md`** — Техническое задание и основные этапы реализации.
* **`userFlow.md`** — Визуализация путей пользователя внутри приложения.
* **`grow.md`** — План развития проекта (Roadmap) и идеи по масштабированию.
* **`optimize.md`** — Стратегия оптимизации производительности и рефакторинга.
* **`pr.md`** — Регламент работы с Pull Request и код-ревью.
* **`review.md`** — Результаты тестирования и отчеты о проверке кода.
* **`remark.md`** — Заметки по улучшению UI/UX и визуальной составляющей.
* **`terms.md`** — Глоссарий терминов и технические стандарты проекта.
* **`pc-configuration.pdf`** — Сводная таблица конфигураций и чертежи сборок.
